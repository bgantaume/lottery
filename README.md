# Lottery

This project demonstrates a lottery game written in Solidity.

## Run the tests

```
npm install
```

```
npx hardhat test
```

## Rules

Each player can play 2eth as many times he wants.
Once in a while the organizer rolls the lottery and a new winners gains 90% of the lottery.
